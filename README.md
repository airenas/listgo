###Build notes

1. Go to base 'listgo' dir
2. Get required packages: 
    'go get ./...'
    'go get github.com/petergtz/pegomock/...'
    'go get github.com/smartystreets/goconvey/convey'
3. Generate mocks: 
    'go generate ./...'
